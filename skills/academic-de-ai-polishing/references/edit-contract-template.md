# Edit Contract Template

Use this template when a polishing request is vague or when the source text, section role, citation boundary, or allowed edit intensity is unclear.

## Target Exemplar Intake

Ask whether the user can provide 3-10 final target examples:

- accepted articles from the target journal
- approved thesis chapters
- supervisor-approved revisions
- rebuttal letters that match the target tone

Use them to infer cadence, section role, level of criticism, and preferred terminology. Do not imitate distinctive phrases or copy protected wording.

## Edit Contract

```text
Source text:
Target language:
Section type:
Paragraph role:
Target exemplar pattern:
Fixed claims:
Fixed citations:
Do-not-change terms:
Unresolved evidence gaps:
Allowed intensity: light | moderate | heavy
Output wanted: diagnosis only | rewrite | rewrite + change note
```
