# Target And Benchmark Writing Alignment

Use this reference before drafting or revising when the user provides 3-10 target papers, benchmark papers, accepted manuscripts, approved theses, or supervisor-approved chapters. The goal is to calibrate section function, evidence density, and claim discipline, not to imitate wording.

## Alignment Brief

```text
Target set:
Section to draft:
Section job in benchmark papers:
Typical evidence density:
Claim style:
Citation pattern:
Figure/table callout pattern:
Limitation and uncertainty style:
Features to match:
Features to avoid:
Drafting implication:
```

## Rules

1. Extract section function before wording: how target papers frame the problem, narrow the gap, justify methods, interpret results, and state limitations.
2. Match evidence density and claim discipline before prose tone.
3. Use target papers to calibrate paragraph roles, citation grouping, figure/table callouts, equation placement, and validation detail.
4. Do not copy distinctive phrasing, sentence rhythm, or proprietary text from target papers.
5. If benchmark patterns conflict with the user's evidence strength, follow the evidence and soften the claim.
6. If no examples exist, use a provisional general academic baseline and list what may need recalibration.

## Drafting Implications

For each target-aligned section, decide:

- what the section must accomplish for this paper;
- which claims can be supported now;
- which citations, data, figures, or formulas are required before drafting;
- which benchmark features should be matched;
- which benchmark features must be avoided because they exceed the current evidence.

When producing the draft, state how it follows the alignment brief and where it intentionally diverges because the user's evidence, audience, or format differs.
