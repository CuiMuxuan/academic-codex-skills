# Writing Output Templates

Use these templates when the user has no school, journal, supervisor, or exemplar manuscript template. If the user provides 3-10 target papers, theses, or final accepted examples, infer section depth, heading style, evidence density, and rhetorical balance from those examples first.

## Target Exemplar Intake

Ask for any available target examples before drafting:

| Field | Notes |
|---|---|
| `target_examples` | 3-10 final papers, theses, accepted manuscripts, or supervisor-approved chapters |
| `target_outlet_or_school` | Journal, conference, school, department, or supervisor |
| `current_field` | Confirmed or inferred research field |
| `target_venue_field` | Field implied by journal, conference, school, or supervisor standard |
| `features_to_match` | Section structure, evidence density, tone, bilingual style, figure/table callouts |
| `features_to_avoid` | Overlong background, excessive citation strings, weak discussion, formulaic language |

If no examples are provided, use the generic templates below and label the baseline as provisional.

## Anti-Generic Writing Checks

Before returning a draft, ask:

| Check | Pass condition |
|---|---|
| Section job | The section has a role inferred from the target outlet or benchmark papers, not just a generic heading. |
| Evidence density | The amount of citation, data, mechanism, and limitation detail is calibrated to the target examples or clearly marked provisional. |
| Claim strength | Novelty and value claims do not exceed the manuscript's evidence, even if benchmark papers make stronger claims. |
| Paragraph function | Each paragraph has a job such as problem framing, gap narrowing, method justification, result interpretation, limitation, or transition. |
| Benchmark use | Benchmark papers shape structure and evidence expectations without copied wording or distinctive phrasing. |
