# Image Panel Integrity

Use this reference for photographs, microscopy, medical images, remote-sensing images, image plates, screenshots used as evidence, or mixed raster/vector figures.

## Source And Permission

Before using image panels, record:

```text
Original image source:
Permission or ownership:
Acquisition or generation method:
Processing performed:
Crop/resize status:
Scale or spatial calibration:
What the image is allowed to prove:
```

Do not use AI-generated or illustrative images as evidence of an experiment, specimen, system output, or measured state.

## Scale, Crop, And Processing

- Include scale bars when magnification, microscopy, spatial size, or physical extent matters.
- Keep scale bars inside the image panel and readable at final size.
- Crop only to improve focus or layout; do not crop out contradictory evidence.
- Apply brightness, contrast, or colour adjustments consistently across comparable images.
- Do not selectively enhance, remove, clone, blur, or obscure features unless the manuscript explicitly justifies it and the venue permits it.
- Keep the original image or source path traceable for audit.

## Image Plate Layout

- Use consistent tile size for comparable images.
- Align panels by condition, time point, sample, or method in a predictable grid.
- Put channel labels and scale bars in consistent positions.
- Use black backgrounds for fluorescence or dark-field conventions; otherwise prefer white or neutral backgrounds.
- Keep gutters visible so adjacent images do not merge.

## Mixed Raster/Vector Figures

- Add labels, arrows, boxes, and scale bars as editable vector overlays when possible.
- Verify embedded raster resolution at final physical size.
- Do not flatten vector labels into a low-resolution raster unless the journal requires one final raster export.
- Keep a layout source file that preserves the raster image and vector annotation layers.

## Screenshots

Screenshots are acceptable only when the interface or visual output itself is evidence.

- Capture at high enough resolution for final size.
- Remove private information.
- Use callout boxes sparingly and consistently.
- Do not use screenshots as substitutes for plots, tables, code-backed diagrams, or text that can be recreated cleanly.

## Output Packet

```text
Original image evidence:
Processing notes:
Scale bar status:
Vector overlay status:
Final raster resolution:
Integrity risks:
```

## Source Basis

This standard follows common image-integrity principles from Nature, PLOS, JCB, and Springer Nature-style figure guidance that emphasizes scale bars, final-size resolution, and non-misleading processing.
