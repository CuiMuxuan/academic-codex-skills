# DOCX Extraction Playbook

Use this reference for DOCX structure, comments, tracked changes, style inspection, captions, and formatting-template inventory.

## Inventory Fields

Record:

- file path;
- file type;
- size;
- document-part count when available;
- likely role: thesis draft, template, review file, appendix, or unknown.

## Extraction Fields

Capture:

- heading hierarchy;
- paragraph index;
- paragraph style;
- table count and rough location;
- figure/table captions;
- comments and authors when available;
- tracked insertions/deletions when available;
- section/page setup signals when inspecting templates;
- extraction quality and notes.

## Quality Review

Check:

- comments not captured;
- tracked changes hidden in XML but not summarized;
- style names missing or inconsistent;
- captions not detected;
- section/page setup signals that require Word verification.

Word automation is optional and must be explicitly approved before use.
