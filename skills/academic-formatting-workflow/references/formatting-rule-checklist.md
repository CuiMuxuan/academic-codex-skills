# Formatting Rule Checklist

Use this checklist when extracting rules from a school handbook, Word template, journal guide, or existing formatted DOCX.

## Rule Table Fields

Capture rules as:

| Field | Meaning |
|---|---|
| `area` | page setup, heading, body, figure, table, reference, appendix |
| `rule` | exact requirement in concise form |
| `source` | handbook page, template style, journal URL, user instruction |
| `priority` | official, user, existing, provisional |
| `automation` | python-docx, Word COM, manual, unknown |
| `status` | pending, applied, needs Word refresh, unresolved |

## High-Risk Areas

Check these before claiming final formatting:

- Section-specific page numbering.
- Table of contents field refresh.
- Linked headers and footers.
- East Asian and Latin font mapping.
- Figure/table caption numbering.
- Reference hanging indent.
- Appendices and attachment pages.
- Mixed portrait and landscape sections.

## General Bilingual Thesis Baseline

Use only when no official guide is available:

- Keep Chinese and English abstracts separate.
- Use consistent keyword labels.
- Use stable heading hierarchy.
- Apply first-line indent to body text if appropriate for the target language.
- Keep captions concise and numbered.
- Keep references in one consistent style.

Always label this baseline as provisional.
