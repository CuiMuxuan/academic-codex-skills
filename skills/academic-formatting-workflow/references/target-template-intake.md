# Target Template Intake

Use this before formatting when the user has not provided a strict handbook or template.

## Preferred Inputs

Ask for 3-10 final target examples, in this priority order:

1. Official school or journal Word template.
2. School formatting handbook or journal author guide.
3. Approved thesis or accepted article from the same target.
4. Supervisor-approved old version.
5. Existing DOCX that the user believes is close to correct.

## What To Extract

| area | examples to inspect | output |
|---|---|---|
| page setup | margins, paper size, sections | rule-table rows |
| headings | numbering, font, spacing | style inventory |
| bilingual front matter | Chinese/English abstract, keywords | checklist |
| captions | figure/table labels and numbering | caption rules |
| references | hanging indent, punctuation, order | reference style note |

If no examples are available, use the provisional general academic baseline in `SKILL.md` and mark all inferred rules as `provisional`.
